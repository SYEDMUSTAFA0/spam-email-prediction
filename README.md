# Spam Email Prediction

A machine learning project that classifies emails as spam or ham (not spam).

## Features
- Email text preprocessing
- Feature extraction using NLP
- Spam detection using ML algorithms
- Model evaluation and prediction

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK
- Jupyter Notebook

## Algorithms Used
- Naive Bayes
- Logistic Regression
- Support Vector Machine (optional)

## Dataset
Uses spam/ham labeled email dataset for training and testing.
